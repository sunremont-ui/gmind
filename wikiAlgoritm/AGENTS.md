# AGENTS.md — MathAlgo Wiki

## Назначение

Вики по математическим алгоритмам: визуализация данных, геометрия деталей машин (редукторы, шестерни), численные методы, вычислительная геометрия, алгоритмы на графах.

## Категории концептов

| Папка | Содержимое |
|-------|-----------|
| `concepts/visualization/` | Алгоритмы визуализации: рендеринг, графики, карты, 3D |
| `concepts/geometry/` | Геометрия деталей: редукторы, шестерни, кулачки, профили |
| `concepts/numerical/` | Численные методы: МКЭ, интерполяция, приближение |
| `concepts/graph/` | Алгоритмы на графах: поиск, пути, потоки |
| `concepts/computational-geometry/` | Вычислительная геометрия: пересечения, триангуляция, выпуклая оболочка |
| `entities/` | Авторы, библиотеки, стандарты |
| `syntheses/` | Перекрёстные анализы, сравнения методов |

## Соглашения

### Язык
- Основной язык: **русский**
- Технические термины: русский + (English) в скобках при первом упоминании
- Формулы: LaTeX inline `$...$` и block `$$...$$`
- Псевдокод: блоки ` ```pseudocode ```  `

### Frontmatter страниц
```yaml
---
title: Название
category: visualization | geometry | numerical | graph | computational-geometry | entity | synthesis
tags: [тег1, тег2]
complexity: O(n log n)   # только для алгоритмов
sources: 0
updated: YYYY-MM-DD
version: 1.0
---
```

### Версионирование страниц
- Поле `version:` в frontmatter — семантическое (1.0, 1.1, 2.0)
- Мажорная версия: переписана структура или добавлен новый алгоритм
- Минорная версия: дополнение, уточнение, новая ссылка
- В `log.md` каждое обновление фиксируется

### Специальные маркеры
- `> ⚠️ Противоречие:` — конфликт между источниками
- `> 📐 Применение:` — конкретный пример применения в геометрии
- `> 🔢 Сложность:` — анализ сложности алгоритма
- `> 🔗 Реализация:` — ссылка на код или библиотеку

## Операции

Поддерживаемые команды (через `/wiki`):
- `/wiki ingest <source>` — добавить источник
- `/wiki query <вопрос>` — запросить знания
- `/wiki lint` — проверить здоровье вики
- `/wiki add <название>` — создать страницу вручную
- `/wiki synth <тема>` — синтез по теме
- `/wiki status` — состояние вики
