# Профиль кулачка (Cam Profile)

## Зачем нужен кулачковый механизм

Кулачковый механизм преобразует равномерное вращение в **произвольный закон движения** толкателя. Это одно из немногих устройств, позволяющих задать любую заданную зависимость перемещения от угла поворота — включая остановки, реверс, нелинейные участки.

**Где это нужно:**
- Механические системы с жёсткими требованиями к кинематике (не программируемые)
- Высокочастотные циклы (тысячи оборотов/мин) — электроника реагирует медленнее
- Надёжность: нет программного обеспечения, нет ошибок прошивки, нет электропитания
- Точное повторение цикла из миллионов операций без деградации

**Типичные применения:**
- Распределительный вал ДВС: открытие/закрытие клапанов по строгому закону
- Автоматы-токарные (swiss-type lathe): до 10 кулачков управляют одновременно резцами, подачей, прутком
- Упаковочные и фасовочные машины: цикл укладки, запайки, обрезки
- Текстильные машины: прокладка нити по сложной траектории
- Полиграфия: механизм захвата листа, кинематика красочного аппарата

## Классификация кулачковых механизмов

### По виду движения толкателя
| Тип | Описание | Применение |
|-----|----------|-----------|
| Поступательный | Толкатель движется по прямой | Клапаны, прессы |
| Качающийся | Толкатель поворачивается вокруг оси | Рычажные механизмы |
| Пространственный | 3D-движение | ЧПУ-фрезерные головки |

### По конструкции ролика
| Тип | Преимущество | Недостаток |
|-----|-------------|-----------|
| С роликом | Малое трение | Минимальный радиус кривизны ≥ радиус ролика |
| С тарельчатым толкателем | Компактность | Высокий износ |
| С острым толкателем | Точность, малые размеры | Только для малых нагрузок |

## Законы движения толкателя

Закон движения $s(\varphi)$ — функция перемещения от угла поворота кулачка. Выбор закона критически влияет на динамику, износ, шум.

### 1. Равномерный (постоянная скорость)
$$s = h \cdot \varphi / \Phi$$

Ускорение = 0, но в начале и конце — **ступенчатый разрыв скорости** → бесконечное ускорение → удар.

**Когда применять**: только в очень тихоходных машинах ($n < 30$ об/мин). Например, простые механические весы.

### 2. Простой гармонический (косинусоидальный)
$$s = \frac{h}{2}\left(1 - \cos\frac{\pi\varphi}{\Phi}\right)$$
$$v = \frac{\pi h}{2\Phi}\sin\frac{\pi\varphi}{\Phi}, \quad a = \frac{\pi^2 h}{2\Phi^2}\cos\frac{\pi\varphi}{\Phi}$$

Ускорение непрерывно, но имеет конечный скачок на краях фазы → **мягкий удар**.

**Когда применять**: средние скорости ($n < 500$ об/мин), умеренные нагрузки, ненагруженные возвраты. Часто используется в бытовой технике.

### 3. Синусоидальный (модифицированный трапециевидный составной)
$$s = h\left(\frac{\varphi}{\Phi} - \frac{1}{2\pi}\sin\frac{2\pi\varphi}{\Phi}\right)$$

Ускорение финитно и непрерывно. Рывок (производная ускорения) конечен на краях.

**Когда применять**: высокие скорости ($n = 500$–$1500$ об/мин), ударонагруженные механизмы.

### 4. Полином 3-4-5 (гладкий переход)
$$s = h\left(10u^3 - 15u^4 + 6u^5\right), \quad u = \varphi/\Phi$$

Скорость и ускорение = 0 в начале и в конце фазы → **нулевой удар**.

Производные:
$$v = h\Phi^{-1}\left(30u^2 - 60u^3 + 30u^4\right)$$
$$a = h\Phi^{-2}\left(60u - 180u^2 + 120u^3\right)$$

**Когда применять**: высокоточные механизмы, прецизионные автоматы, медицинское оборудование. Золотой стандарт для новых конструкций.

### 5. Модифицированный трапециевидный
Кусочная функция: сначала постоянное ускорение, потом косинусный переход, потом постоянное замедление.

Максимальное ускорение минимально среди всех законов с нулевой начальной/конечной скоростью.

**Когда применять**: автоматы-токарные с большими инерционными массами, фасовочные машины (минимизация нагрузки на привод).

## Алгоритм построения профиля кулачка

### Метод обращения движения

Ключевая идея: представим, что кулачок неподвижен, а толкатель вращается вокруг оси кулачка в обратном направлении. Тогда ось толкателя описывает кривую — это **линия центров** роликов. Профиль кулачка — эквидистанта этой кривой на расстоянии $r_{roller}$ внутрь.

```pseudocode
function buildCamProfile(law, r_base, e, roller_r, n_points):
    // law: функция s(phi) — закон движения
    // r_base: базовый радиус кулачка
    // e: эксцентриситет (смещение оси толкателя от оси кулачка, 0 для центрального)
    // roller_r: радиус ролика

    phi = linspace(0, 2*PI, n_points)
    roller_centers = []

    for each phi_i:
        s = law(phi_i)
        r = r_base + s          // расстояние от оси до центра ролика

        // Для центрального поступательного толкателя:
        if e == 0:
            x = r * cos(phi_i)
            y = r * sin(phi_i)
        else:
            // Со смещённым толкателем — сложнее:
            psi = arcsin(e/r)    // угол смещения
            x = r * cos(phi_i + psi)
            y = r * sin(phi_i + psi)

        roller_centers.append((x, y))

    // Эквидистанта: сдвинуть каждую точку внутрь на roller_r по нормали
    cam_profile = []
    for i in 0..n_points:
        tangent = roller_centers[(i+1)%n] - roller_centers[(i-1)%n]
        normal = normalize((-tangent.y, tangent.x))  // нормаль внутрь
        cam_point = roller_centers[i] - roller_r * normal
        cam_profile.append(cam_point)

    return cam_profile
```

### Проверка на заострение

Если радиус кривизны профиля кулачка меньше радиуса ролика, профиль самопересекается — **заострение** (undercutting). Деталь нельзя изготовить.

```pseudocode
function checkUndercutting(s_func, r_base, n_points):
    // Радиус кривизны профиля для поступательного толкателя:
    for phi in linspace(0, 2*PI, n_points):
        r  = r_base + s_func(phi)
        r1 = ds/dphi          // первая производная
        r2 = d2s/dphi2        // вторая производная
        rho = (r^2 + r1^2)^(3/2) / abs(r^2 + 2*r1^2 - r*r2)
        if rho < roller_r:
            return False, phi  // заострение при угле phi
    return True, None
```

При обнаружении заострения: увеличить $r_0$ или уменьшить $r_{roller}$.

## Давление передачи и угол давления

**Угол давления** $\mu$ — угол между нормалью к профилю и направлением движения толкателя. При больших $\mu$ растут боковые нагрузки на направляющие.

$$\tan\mu = \frac{ds/d\varphi - e}{\sqrt{(r_0 + s)^2 - e^2}}$$

Рекомендации: $\mu_{max} < 30°$ для поступательного, $\mu_{max} < 45°$ для качающегося.

Снижение $\mu$: увеличить $r_0$, уменьшить эксцентриситет $e$, изменить закон движения.

## Динамика кулачкового механизма

Вибрации возникают когда частота возбуждения (от закона движения) близка к собственной частоте системы толкатель-пружина.

**Коэффициент динамичности**: $Q = \omega^2 / \omega_0^2$ — отношение квадратов частот.

При $Q > 0.1$ — нужен динамический анализ. При $Q > 0.5$ — обязателен.

Пружина удержания толкателя должна обеспечивать контакт при максимальном отрицательном ускорении:
$$c \cdot (x_{пр} + s) > m \cdot |a_{max}| + F_{трения}$$

## Применение по областям

### Двигатели внутреннего сгорания
- **Распредвал**: 2–4 кулачка на цилиндр (впуск + выпуск), $n$ до 8000 об/мин. Закон движения — компромисс между наполнением цилиндра (нужно долгое открытие) и динамикой (нужно плавное ускорение).
- **Фазовращатели (VVT)**: поворот распредвала гидравликой → изменение фазы открытия клапана на ходу. Кулачок тот же, но смещён.
- **Профиль спортивных кулачков**: более агрессивный (больший ход, дольше открыт) — за счёт ресурса и плавности.

### Автоматы-токарные (Swiss-type lathe)
- До 10–15 кулачков на одном кулачковом барабане
- Каждый управляет своим инструментом (резец, сверло, отрезной нож, подача прутка)
- Синхронизация: все на одном валу → гарантированная синхронность
- Сменные кулачки под каждую деталь → переналадка за часы
- Применяется для массового производства часовых деталей, медицинских имплантов, коннекторов

### Упаковочное оборудование
- **Укладочные головки**: кулачок управляет захватом — плавный подхват без удара о продукт
- **Термосварочные губки**: синхронизация момента зажима с циклом пленки
- **Дозаторы**: поршневой насос с кулачковым приводом — закон движения = профиль расхода
- Требование: безударный закон (полином 3-4-5) из-за деликатного продукта (печенье, таблетки)

### Текстильное производство
- **Вязальные машины**: кулачки управляют иглами (до 36 кулачков на полотно)
- **Ткацкие станки (раппие)**: кулачок прокладывания уточной нити по сложной траектории обхода нитей основы
- Тысячи оборотов/час → строжайшие требования к закону движения и прочности

### Медицинское оборудование
- **Перистальтические насосы**: кулачки сжимают трубку последовательно → постоянный поток без клапанов, стерильно
- **Дозирующие устройства для таблеток**: кулачок управляет лотком подачи, пуансоном прессования

### Принтеры и копировальная техника
- **Механизм захвата листа**: кулачок отделяет верхний лист из стопки (дифференцированное трение + кулачковый подпружиненный ролик)
- **Фьюзерный прижим**: кулачок задаёт нагрузку на тефлоновый вал в зависимости от типа бумаги
