# Конические зубчатые колёса (Bevel Gears)

## Зачем нужны конические передачи

Цилиндрические шестерни передают вращение между **параллельными** осями. Конические колёса — между **пересекающимися** осями. Это не взаимозаменяемые конструкции: там, где нужен поворот оси вращения на 90°, конические колёса — единственное стандартное зубчатое решение.

**Где неизбежно нужны:**
- Главная передача автомобиля (продольный вал КПП → поперечные полуоси)
- Угловые редукторы (поворот оси привода без потери жёсткости)
- Дифференциалы (равномерное распределение крутящего момента)
- Вертолётные трансмиссии (горизонтальный ротор от вертикального двигателя)

## Геометрия: конусы и углы

Базовая концепция: две конические поверхности катятся без скольжения (делительные конусы). В точке касания конусов — полюс зацепления.

Для прямого угла между осями ($\Sigma = 90°$):
$$\delta_1 + \delta_2 = 90°, \quad \tan\delta_1 = z_1/z_2, \quad \tan\delta_2 = z_2/z_1$$

Для произвольного угла $\Sigma$:
$$\tan\delta_1 = \frac{\sin\Sigma}{z_2/z_1 + \cos\Sigma}$$

### Конусные расстояния и параметры

| Параметр | Обозначение | Формула |
|----------|-------------|---------|
| Внешнее конусное расстояние | $R_e$ | $R_e = d_{e2}/(2\sin\delta_2) = m_e z_2/(2\sin\delta_2)$ |
| Ширина венца | $b$ | $b \leq R_e/3$ (нагрузочная способность); $b \leq 10m_e$ |
| Среднее конусное расстояние | $R_m$ | $R_m = R_e - b/2$ |
| Средний модуль | $m_m$ | $m_m = m_e(1 - b/(2R_e))$ |
| Внешний диаметр вершин | $d_{ae}$ | $d_{ae} = m_e(z + 2\cos\delta)$ |
| Угол конуса вершин | $\delta_a$ | $\delta_a = \delta + \theta_a$, $\theta_a = \arctan(1/z)$ |

## Типы конических колёс

### Прямозубые (Straight Bevel)
Линия зуба радиальная (лежит на образующей конуса).

```pseudocode
function buildStraightBevelTooth(me, z, delta_deg, n_pts):
    delta = deg2rad(delta_deg)
    Re = me * z / (2*sin(delta))
    Rm = Re - b/2
    mm = me * Rm / Re     // средний модуль
    zv = z / cos(delta)   // число зубьев эквивалентного цилиндрического колеса

    // Профиль в среднем сечении = эвольвентный профиль с mm и zv:
    profile_2d = buildEvolvente(mm, zv)

    // Масштабировать по конусу от Re до Re-b:
    tooth_3d = []
    for u in linspace(0, b, n_pts):
        scale = (Re - u) / Re
        section = scaleProfile(profile_2d, scale)
        section = rotateOnCone(section, delta, u)
        tooth_3d.extend(section)
    return tooth_3d
```

**Плюсы**: простое изготовление, дешевле.
**Минусы**: шумнее, меньшая нагрузочная способность, осевое усилие.

**Применение**: малые нагрузки, тихоходные передачи, бытовая техника, ручные инструменты.

### Спиральные (Spiral Bevel / Hypoid)
Линия зуба — спираль (дуга окружности или эвольвента на конусе). Угол наклона $\beta_m = 25°$–$45°$.

Типы систем нарезания:
- **Gleason**: дуга окружности. США, широко распространён.
- **Klingelnberg (Palloid)**: эвольвента. Германия, более равномерный контакт.
- **Oerlikon**: эпициклоида.

> 🔢 Степень перекрытия спиральных зубьев $\varepsilon_\beta = b\sin\beta_m / (\pi m_m)$ добавляется к торцевому $\varepsilon_\alpha$ → суммарная $\varepsilon_\gamma = \varepsilon_\alpha + \varepsilon_\beta > 2$ → очень плавное зацепление.

**Плюсы**: высокая нагрузочная способность, тихая работа, высокий КПД.
**Минусы**: требует точного монтажа (осевой люфт → смещение пятна контакта), сложнее в изготовлении.

**Применение**: главные передачи легковых автомобилей, авиационные редукторы, высоконагруженные промышленные передачи.

### Гипоидные (Hypoid)
Оси не пересекаются — смещены на расстояние $E$ (offset). Смещение позволяет:
- Опустить карданный вал ниже пола кузова (центр тяжести ниже)
- Увеличить передаточное отношение в одной паре

КПД ниже спиральных из-за дополнительного скольжения. Требуют специальных гипоидных масел (EP-добавки).

**Применение**: **главные передачи большинства легковых автомобилей** (задний и полный привод) — смещение оси карданного вала вниз освобождает место для пола салона.

## Нарезание конических колёс

### Строгание (Planing / Generating)
Два прямолинейных резца воспроизводят зубья плоского производящего колеса (crown gear). Для прямозубых.

### Метод обкатки с дисковым резцом (Face-milling)
Дисковая резцовая головка нарезает впадины одну за другой — метод для крупносерийного производства (Gleason).

### Метод непрерывного обкатки (Face-hobbing)
Резцовая головка и заготовка вращаются синхронно → непрерывное нарезание (Klingelnberg, Oerlikon). Для серийного производства.

### Шлифование
- Специализированные станки Gleason/Klingelnberg
- Достигается 4–5 степень точности по DIN 3965
- Обязательно для тихоходных авиационных редукторов

## Расчёт пятна контакта

Правильная работа конической передачи критически зависит от расположения **пятна контакта** (contact pattern). При монтаже проверяют прокатку с красящей пастой.

Идеальное пятно: центрировано по длине зуба, занимает $55$–$65\%$ длины, расположено в центре по высоте.

```pseudocode
function checkContactPattern(gear_model, load_N):
    // Упругое смещение зубьев под нагрузкой:
    deflection = FEA(gear_model, load_N)

    // Смещение пятна под нагрузкой предсказывается по жёсткости зуба:
    // Пятно должно сместиться к середине длины, не к краям
    pattern_center = computeContactEllipse(deflection)

    if pattern_center.x > 0.7*b or pattern_center.x < 0.3*b:
        return "BAD: edge contact — зуб перекосится"
    return "OK"
```

## Применение по областям

### Автомобилестроение
- **Главная передача** (rear axle): гипоидная пара, $i = 2.9$–$4.1$, момент до 3000 Н·м. Ресурс > 300 000 км.
- **Дифференциал**: конические сателлиты. Прямозубые или слегка спиральные. Распределяют момент при разных скоростях колёс.
- **Раздаточная коробка полного привода**: спиральная или гипоидная. Передаёт момент на переднюю ось.
- **Рулевые колонки** угловые приводы: небольшие спиральные пары.

### Авиация
- **Вертолётные главные редукторы**: спиральные конические + планетарные. Ми-8: 4 конические пары в главном редукторе. $n_{ротора} = 192$ об/мин от $n_{двигателя} = 19000$ об/мин.
- **Приводы винтов**: вал двигателя горизонтальный → вал винта вертикальный (вертолёт) или наклонный.
- **Вспомогательные агрегаты** (AGB — accessory gearbox): конические пары привода генераторов, гидронасосов.

### Станкостроение
- **Угловые головки** для обработки поднутрений: фрезерный шпиндель поворачивается на 90° через конический редуктор.
- **Делительные устройства**: угловые приводы с точными коническими парами (5–6 степень точности).
- **Ниткошлифовальные станки**: кинематическая цепь с коническими парами для формирования сложных профилей.

### Морской транспорт
- **Z-приводы (Azimuth thruster)**: электродвигатель → угловой конический редуктор → гребной винт, всё устройство поворачивается → функция руля. Применяется на судах DP-класса, паромах.
- **Угловые приводы насосов**: гидравлические насосы с вертикальной осью от горизонтального вала.

### Промышленность
- **Угловые редукторы конвейеров**: поворот линии транспортировки на 90°
- **Миксеры и мешалки**: вертикальная мешалка от горизонтального привода
- **Горнодобывающие машины**: угловые редукторы приводов скребков и транспортёров
