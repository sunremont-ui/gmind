# Эвольвентное зубчатое зацепление (Involute Gear)

## Зачем эвольвентный профиль

Задача зубчатой передачи: передать вращение с постоянным **передаточным отношением**. Это означает, что мгновенная скорость вращения ведомого колеса должна быть строго пропорциональна скорости ведущего в любой момент времени — иначе возникают вибрации, шум, нагрузки, разрушающие передачу.

По **теореме Виллиса**: для постоянного передаточного отношения общая нормаль к профилям зубьев в точке контакта должна всегда проходить через фиксированную точку на линии центров — **полюс зацепления**.

**Почему именно эвольвента?**
- Единственная кривая, где нормаль к профилю — касательная к основной окружности. Это автоматически гарантирует полюс зацепления.
- Линия зацепления (путь точки контакта) — прямая. Это упрощает динамику сил.
- **Взаимозаменяемость**: любые два колеса с одинаковым модулем зацепляются правильно, независимо от числа зубьев. Это революционное свойство, позволяющее стандартизировать производство.
- Технологичность: нарезается стандартным реечным инструментом (долбяком, червячной фрезой) при одном базовом профиле.

Исторически использовали циклоидный профиль (часовые механизмы). Эвольвентный победил в машиностроении из-за взаимозаменяемости и простоты инструмента.

## Геометрия эвольвенты

Эвольвента окружности — кривая, описываемая концом нити при разматывании с цилиндра. Параметрические уравнения эвольвенты с основной окружностью радиуса $r_b$:

$$x(\theta) = r_b (\cos\theta + \theta \sin\theta)$$
$$y(\theta) = r_b (\sin\theta - \theta \cos\theta)$$

Ключевые свойства:
- Нормаль к эвольвенте в любой точке касательна к основной окружности
- Радиус кривизны в точке = длине касательной к основной окружности
- Эвольвента начинается на основной окружности ($\theta = 0$) и уходит наружу

## Основные параметры

| Параметр | Обозначение | Формула | Примечание |
|----------|-------------|---------|-----------|
| Модуль | $m$ | — | Стандартные ряды: 1, 1.25, 1.5, 2, 2.5, 3, 4, 5... |
| Число зубьев | $z$ | — | $z \geq z_{min}$ для отсутствия подрезания |
| Угол профиля | $\alpha$ | 20° | Стандарт ГОСТ, DIN, ISO |
| Делительный диаметр | $d$ | $d = mz$ | Основной размер |
| Основной диаметр | $d_b$ | $d_b = d\cos\alpha$ | База эвольвенты |
| Диаметр вершин | $d_a$ | $d_a = m(z+2)$ | По вершинам зубьев |
| Диаметр впадин | $d_f$ | $d_f = m(z-2.5)$ | По впадинам |
| Шаг | $p$ | $p = \pi m$ | Дуга делительной окружности |
| Основной шаг | $p_b$ | $p_b = p\cos\alpha$ | Постоянен для данного $m$ и $\alpha$ |

## Алгоритм построения профиля зуба

```pseudocode
function buildToothProfile(m, z, alpha_deg=20, n_points=100):
    alpha = deg2rad(alpha_deg)
    r  = m * z / 2          // делительный радиус
    rb = r * cos(alpha)     // основной радиус
    ra = r + m              // радиус вершин
    rf = r - 1.25*m         // радиус впадин

    // Угол эвольвентного параметра на делительной окружности:
    theta_pitch = sqrt((r/rb)^2 - 1)  // = tan(alpha)

    // Угол на вершинной окружности:
    theta_tip = sqrt((ra/rb)^2 - 1)

    // Построение правой стороны зуба:
    profile_right = []
    for i = 0 to n_points:
        theta = i * theta_tip / n_points
        x = rb * (cos(theta) + theta * sin(theta))
        y = rb * (sin(theta) - theta * cos(theta))
        profile_right.append((x, y))

    // Зеркально отразить для левой стороны
    // + дуга по впадине (окружность радиуса rf)
    // + дуга по вершине (окружность радиуса ra)

    return assembleFullTooth(profile_right, rb, ra, rf, z)

function buildGear(m, z, alpha_deg=20):
    tooth = buildToothProfile(m, z, alpha_deg)
    gear = []
    for i = 0 to z-1:
        angle = 2*PI*i/z
        gear.extend(rotate(tooth, angle))
    return gear
```

## Условие зацепления и степень перекрытия

Для правильного зацепления двух колёс ($z_1$, $z_2$, один модуль $m$):

**Межосевое расстояние**: $a = m(z_1 + z_2)/2$

**Степень перекрытия** — среднее число пар зубьев в зацеплении:
$$\varepsilon_\alpha = \frac{\sqrt{r_{a1}^2 - r_{b1}^2} + \sqrt{r_{a2}^2 - r_{b2}^2} - a\sin\alpha}{\pi m \cos\alpha}$$

- $\varepsilon_\alpha < 1$ — недопустимо: часть периода без контакта → удар
- $1 < \varepsilon_\alpha < 2$ — одна или две пары в контакте попеременно
- $\varepsilon_\alpha > 2$ — всегда минимум две пары → плавнее, шумит меньше

**Стандартная рекомендация**: $\varepsilon_\alpha \geq 1.2$.

## Подрезание зуба

При малом числе зубьев реечный инструмент срезает ножку эвольвентного профиля — **подрезание** ослабляет зуб и уменьшает степень перекрытия.

$$z_{min} = \frac{2h_a^*}{\sin^2\alpha} = \frac{2 \cdot 1}{\sin^2 20°} \approx 17$$

При $z < 17$: использовать **корригирование** (смещение исходного контура $x > 0$). Смещение сдвигает зуб наружу, устраняя подрезание.

**Коэффициент смещения**: $x_{min} = 1 - z\sin^2\alpha/2$

Корригированные передачи позволяют использовать $z = 10$–$12$ при $\alpha = 20°$.

## Силы в зацеплении

Нормальная сила $F_n$ направлена под углом $\alpha$ к касательной к делительной окружности.

Разложение на цилиндрическом колесе:
- Окружная: $F_t = 2T/d$ ($T$ — крутящий момент)
- Радиальная: $F_r = F_t \tan\alpha$
- Нормальная: $F_n = F_t / \cos\alpha$

Для косозубых колёс (угол наклона $\beta$):
- Осевая: $F_a = F_t \tan\beta$

## Нарезание и стандарты

### Способы нарезания
- **Фрезерование**: дисковая или пальцевая фреза с профилем впадины. Дёшево, но точность низкая (зависит от числа зубьев).
- **Обкатка (hobbing)**: червячная фреза одновременно вращается и перемещается. Стандартный метод серийного производства. Один инструмент — все числа зубьев одного модуля.
- **Долбление**: долбяк (зубчатое колесо-инструмент) обкатывается по заготовке. Для ступенчатых валов, внутренних зубьев.
- **Шлифование**: финишная операция для 5–6-й степени точности.

### Стандарты
| Стандарт | Страна | Примечание |
|----------|--------|-----------|
| ГОСТ 13755-81 | РФ/СНГ | Исходный контур, углы |
| ISO 53 | Международный | Основа большинства национальных |
| DIN 867 | Германия | |
| AGMA 201 | США | Дюймовая система (диаметральный питч) |

## Применение по областям

### Автомобилестроение
- **Коробка передач**: косозубые колёса (шумят меньше прямозубых) с $m = 2$–$4$ мм
- **Дифференциал**: конические колёса (см. [[concepts/geometry/bevel-gear]])
- **Рулевая рейка**: шестерня + рейка с переменным передаточным числом (переменный модуль вдоль рейки)
- **Стартер**: прямозубые, большой модуль, малые числа зубьев

### Авиация и космос
- **Авиационные редукторы**: 6-я степень точности и выше, специальные масла. Легкосплавные корпуса, плавающие венцы для равномерного распределения нагрузки.
- **Планетарные редукторы двигателей** (GTF — geared turbofan): большое передаточное отношение при малых габаритах. Ролс-Ройс, Pratt & Whitney.

### Промышленность
- **Шаровые мельницы, конвейеры**: открытые передачи, $m = 10$–$40$ мм, смазка нанесением
- **Редукторы общего назначения** (Ц2, Ц3, КЦ): стандартизированные блоки $i = 8$–$500$
- **Подъёмные краны**: с тормозными колодками, расчёт на удар при пуске

### Роботы и точная механика
- **Сервоприводы**: предварительно нагруженные (preloaded) пары для нулевого люфта
- **Гармонические передачи (Harmonic Drive)**: отношение до 300:1, нулевой люфт — применяется в манипуляторах, экзоскелетах
- **Приборные передачи**: $m = 0.05$–$0.3$ мм, латунь/пластик, часы, счётчики

### Энергетика
- **Ветрогенераторы**: мультипликатор 1:100 для раскрутки генератора от медленной лопасти. Планетарная схема, $m$ до 40 мм, строгие требования по виброакустике.
- **Гидротурбины**: редукторы питательных насосов

### Станкостроение
- **Шпиндельные коробки**: несколько ступеней, переключение → диапазон скоростей
- **ЧПУ-приводы подачи**: рейка + шестерня для линейного перемещения суппорта

## Расчёт напряжений (метод AGMA/ISO)

Два критерия:
1. **Контактная прочность** (Hertz): $\sigma_H \leq [\sigma_H]$
   $$\sigma_H = Z_E Z_H Z_\varepsilon \sqrt{F_t K_A K_v K_{H\beta} K_{H\alpha} / (b d_1)}$$
2. **Изгибная прочность** у корня зуба:
   $$\sigma_F = F_t K_A K_v K_{F\beta} K_{F\alpha} Y_F Y_S / (b m) \leq [\sigma_F]$$

Для точного расчёта корневых напряжений применяется МКЭ (см. [[concepts/numerical/fem-basics]]) с 2D-сечением зуба.
