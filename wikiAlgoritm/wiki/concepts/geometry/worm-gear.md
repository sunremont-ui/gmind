# Червячная пара (Worm Gear)

## Зачем нужна червячная передача

Червячная передача решает задачу, с которой не справляются цилиндрические и конические колёса: **большое передаточное отношение в компактном корпусе** при пересекающихся осях.

Цилиндрические редукторы требуют несколько ступеней для $i > 10$, каждая ступень — пара колёс, корпус, подшипники. Червячная передача даёт $i = 8$–$100$ в **одной ступени**, при этом:
- Компактность: одна ступень вместо двух-трёх
- Плавность: линия контакта скользящая → низкий шум и вибрации
- Самоторможение: ведомое колесо не может вращать ведущий червяк → используется как тормоз

**Главный недостаток** — низкий КПД ($\eta = 0.7$–$0.9$) из-за скольжения витков. Поэтому для мощных приводов (>20 кВт) предпочитают планетарные редукторы.

## Геометрия и типы червяков

### Архимедов червяк (ZA)
Осевое сечение: трапеция с прямыми боковыми сторонами. Боковая поверхность — винтовой геликоид.

$$z_{червяк}(r, \varphi) = \frac{p}{2\pi}\varphi = \frac{m \cdot q}{2} \cdot \arctan(r / r_1) \cdot \frac{p}{\pi}$$

Нарезается прямолинейным резцом в осевой плоскости. Дёшево, точность 8–9 степень.

### Эвольвентный червяк (ZI)
В нормальном сечении — эвольвента. Шлифуется плоским кругом → высокая точность (5–6 степень). Стандарт для силовых передач.

### Тороидальный/глобоидный червяк (ZT)
Ось червяка вогнутая (тор). Одновременно в контакте $3$–$5$ зубьев (против $1$–$2$ у обычных) → нагрузочная способность в 2–3 раза выше. Сложно в изготовлении.

## Основные параметры

| Параметр | Обозначение | Значения/формулы |
|----------|-------------|-----------------|
| Осевой модуль | $m$ | Стандартный ряд: 1..40 |
| Число заходов | $z_1$ | 1, 2, 4 (реже 3) |
| Число зубьев колеса | $z_2$ | $z_2 = i \cdot z_1$ |
| Коэффициент диаметра | $q$ | $q = d_1/m$, стандарт: 8..16 |
| Делительный диаметр червяка | $d_1$ | $d_1 = qm$ |
| Делительный диаметр колеса | $d_2$ | $d_2 = z_2 m$ |
| Межосевое расстояние | $a_w$ | $a_w = (d_1 + d_2)/2$ |
| Угол подъёма витка | $\gamma$ | $\tan\gamma = z_1/q$ |
| Передаточное отношение | $i$ | $i = z_2/z_1$ |

## КПД и самоторможение

КПД прямого хода (червяк — ведущий):
$$\eta = \frac{\tan\gamma}{\tan(\gamma + \rho')}$$

КПД обратного хода (колесо — ведущий):
$$\eta' = \frac{\tan(\gamma - \rho')}{\tan\gamma}$$

где $\rho' = \arctan(\mu / \cos\alpha_n)$ — приведённый угол трения, $\mu$ — коэффициент трения.

**Самоторможение** наступает при $\gamma \leq \rho'$, то есть $\eta' \leq 0$:
$$i_{самоторм} \geq \frac{q}{\tan\rho'} = \frac{q}{\mu/\cos\alpha_n}$$

Типично при $z_1 = 1$: $\gamma \approx 5$–$7°$, $\rho' \approx 4$–$8°$ → самоторможение при $\mu \geq 0.07$.

> ⚠️ Противоречие: самоторможение ненадёжно при вибрациях (разрушает масляная плёнка, трение падает) и при нагреве передачи в работе. Для грузоподъёмных механизмов требуется отдельный тормоз.

## Алгоритм построения витка червяка (ZA)

```pseudocode
function buildArchimedesWorm(m, q, z1, n_axial, n_circ):
    // Основные параметры:
    d1  = q * m            // делительный диаметр
    da1 = d1 + 2*m         // диаметр вершин
    df1 = d1 - 2.4*m       // диаметр впадин
    p_ax = PI * m          // осевой шаг
    lead = p_ax * z1       // ход (перемещение за оборот)
    gamma = atan(z1 / q)   // угол подъёма

    // 3D-поверхность витка — параметрическая форма:
    // u — положение по оси (0..1), phi — угол поворота
    surface_points = []
    for u in linspace(0, z1 * p_ax, n_axial):  // длина нарезки
        for phi in linspace(0, 2*PI, n_circ):
            z = u                           // осевая координата
            r = d1/2                        // боковая поверхность на делительном радиусе
            x = r * cos(phi)
            y = r * sin(phi)
            // Осевое смещение = шаг * угол / (2*PI):
            z_helix = phi * p_ax * z1 / (2*PI)
            surface_points.append((x, y, z + z_helix))

    // Профиль в осевом сечении (трапеция):
    alpha = deg2rad(20)
    flank_points_left  = helix(da1/2, df1/2, alpha, +1)
    flank_points_right = helix(da1/2, df1/2, alpha, -1)

    return surface_points, flank_points_left, flank_points_right
```

## Расчёт температуры и смазка

Из-за скольжения выделяется тепло. Тепловой расчёт:

$$T_{масло} = T_{окр} + \frac{P(1 - \eta)}{K_T \cdot A}$$

где $K_T = 10$–$17$ Вт/(м²·°C) — коэффициент теплоотдачи, $A$ — площадь корпуса.

При $T_{масло} > 95°C$ → принудительное охлаждение (ребра, вентилятор, маслоохладитель).

**Материалы пары**: бронза (колесо) + закалённая сталь (червяк). Бронза мягче, принимает износ на себя, хорошо прирабатывается.

## Применение по областям

### Подъёмно-транспортные машины
- **Лифты малой мощности** (до 5 кВт): самоторможение — аварийный тормоз при отключении питания. Груз не падает без электродвигателя.
- **Ручные тали**: $i = 40$–$80$, малое усилие на рукоятке → большая подъёмная сила
- **Якорные лебёдки**: компактность важна на судне, самоторможение удерживает якорь
- **Секционные ворота**: привод воротных секций, самоторможение держит полотно

### Станкостроение
- **Делительные головки**: $i = 40$, жёсткость и минимальный люфт. Точность деления заготовки.
- **Поворотные столы ЧПУ**: планетарно-червячные механизмы, $i = 90$–$180$, предварительный натяг для устранения люфта.
- **Привод зажимных устройств**: самоторможение удерживает деталь при выключенном приводе

### Автомобилестроение
- **Рулевые механизмы** (устаревшие): глобоидный червяк + ролик — применялся до распространения реечного рулевого. До сих пор используется в грузовиках.
- **Стеклоподъёмники**: малый червячный редуктор в моторе стеклоподъёмника, самоторможение удерживает стекло.
- **Приводы сидений**: регулировка положения, самоторможение фиксирует позицию.

### Пищевое и упаковочное производство
- **Мясорубки**: высокий момент при малой скорости
- **Тестомесы**: постоянная нагрузка при низкой скорости
- **Конвейеры**: компактные мотор-редукторы с встроенным червячным редуктором (SEW-EURODRIVE, Nord, Watt Drive)

### Аэрокосмос
- **Управляющие поверхности** (исторически): рули высоты самолётов до появления гидравлики. Самоторможение — безопасность при обрыве троса.
- **Приводы солнечных панелей** спутников: точное позиционирование, самоторможение экономит питание.
- **Антенные механизмы**: большое передаточное отношение для точного наведения.

### Медицинское оборудование
- **Хирургические столы**: электропривод с самоторможением — стол не опускается при аварии питания
- **Ортопедические кресла**: регулировка угла наклона
