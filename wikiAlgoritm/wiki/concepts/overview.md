# MathAlgo Wiki — Обзор

Вики по математическим алгоритмам с акцентом на практическое применение в инженерии и компьютерной графике.

## Предметные области

### 📊 Визуализация данных (Visualization)
Алгоритмы для построения и рендеринга графических примитивов, кривых и поверхностей.

Ключевые темы:
- [[concepts/visualization/bezier-curves]] — параметрические кривые, управляющие точки
- [[concepts/visualization/spline-interpolation]] — гладкие кривые через точки (B-spline, NURBS)
- [[concepts/visualization/rendering-3d]] — проекции, видимость, освещение

### ⚙️ Геометрия деталей (Geometry)
Математические модели для проектирования деталей редукторов и зубчатых передач.

Ключевые темы:
- [[concepts/geometry/involute-gear]] — эвольвентный профиль зуба, геометрический расчёт
- [[concepts/geometry/cam-profile]] — кинематика кулачкового механизма
- [[concepts/geometry/worm-gear]] — червячная пара, осевые сечения
- [[concepts/geometry/bevel-gear]] — конические колёса, делительный конус

### 🔢 Численные методы (Numerical)
Дискретные методы решения задач математики и механики.

Ключевые темы:
- [[concepts/numerical/fem-basics]] — метод конечных элементов, аппроксимация
- [[concepts/numerical/interpolation]] — восстановление функции по узлам
- [[concepts/numerical/numerical-integration]] — квадратурные формулы
- [[concepts/numerical/linear-algebra]] — эффективное решение систем уравнений

### 📐 Вычислительная геометрия (Computational Geometry)
Алгоритмы для работы с геометрическими объектами.

Ключевые темы:
- [[concepts/computational-geometry/convex-hull]] — построение выпуклой оболочки
- [[concepts/computational-geometry/triangulation]] — разбиение на треугольники
- [[concepts/computational-geometry/boolean-ops]] — пересечение, объединение полигонов

### 🕸️ Алгоритмы на графах (Graph)
Классические задачи на графах с приложениями в CAD и маршрутизации.

Ключевые темы:
- [[concepts/graph/shortest-path]] — Dijkstra, A*
- [[concepts/graph/spanning-tree]] — Kruskal, Prim
