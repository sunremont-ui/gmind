# МКЭ — Метод конечных элементов (Finite Element Method)

Численный метод решения дифференциальных уравнений в частных производных путём разбиения области на конечные элементы.

## Идея метода

1. Разбить область $\Omega$ на конечные элементы $\Omega_e$ (треугольники, тетраэдры, гексаэдры)
2. Аппроксимировать решение на каждом элементе полиномом
3. Свести к системе алгебраических уравнений $\mathbf{K}\mathbf{u} = \mathbf{f}$
4. Решить систему

## Глобальная система уравнений

$$\mathbf{K}\mathbf{u} = \mathbf{f}$$

- $\mathbf{K}$ — матрица жёсткости (stiffness matrix), разрежённая, симметричная
- $\mathbf{u}$ — вектор узловых перемещений
- $\mathbf{f}$ — вектор узловых сил

## Матрица жёсткости треугольного элемента (плоское напряжение)

Для треугольного элемента с линейными функциями формы:

$$\mathbf{K}_e = t \cdot A_e \cdot \mathbf{B}^T \mathbf{D} \mathbf{B}$$

где:
- $t$ — толщина
- $A_e$ — площадь элемента
- $\mathbf{B}$ — матрица деформаций (производные функций формы)
- $\mathbf{D}$ — матрица упругости материала

```pseudocode
function assembleStiffness(nodes, elements, E, nu):
    n_dof = len(nodes) * 2
    K = zeros(n_dof, n_dof)

    D = elasticityMatrix(E, nu)  # 3x3 для плоского напряжения

    for elem in elements:
        n1, n2, n3 = elem.nodes
        Ke = triangleStiffness(nodes[n1], nodes[n2], nodes[n3], D)
        # Сборка в глобальную матрицу (direct stiffness)
        assembleElement(K, Ke, [n1*2, n1*2+1, n2*2, n2*2+1, n3*2, n3*2+1])

    return K
```

## Применение к деталям редукторов

> 📐 Применение: МКЭ используется для расчёта напряжений у основания зуба шестерни (см. [[concepts/geometry/involute-gear]]). Типовая схема: 2D-сечение зуба, граничные условия — защемление по впадине, нагрузка на вершине.
