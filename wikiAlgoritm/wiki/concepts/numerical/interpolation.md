# Интерполяция (Interpolation)

Построение функции, проходящей через заданные узловые точки $(x_i, y_i)$.

## Методы интерполяции

### 1. Полином Лагранжа

$$L(x) = \sum_{i=0}^{n} y_i \prod_{j \neq i} \frac{x - x_j}{x_i - x_j}$$

> 🔢 Сложность: O(n²) для вычисления в одной точке  
> ⚠️ Эффект Рунге: при большом $n$ и равномерных узлах возникают осцилляции на краях.

### 2. Разделённые разности Ньютона

Эффективная форма для добавления новых узлов:

$$N(x) = [y_0] + [y_0, y_1](x-x_0) + [y_0,y_1,y_2](x-x_0)(x-x_1) + \ldots$$

> 🔢 Сложность: O(n²) построение таблицы, O(n) добавление нового узла

### 3. Кубический сплайн

Кусочно-кубическая функция с $C^2$-гладкостью.

```pseudocode
function cubicSpline(x[], y[]):
    n = len(x) - 1
    h[i] = x[i+1] - x[i]

    # Трёхдиагональная система для вторых производных:
    A = tridiagonalMatrix(h)
    b = rightHandSide(y, h)
    m = solveTridiagonal(A, b)  # O(n)

    # Коэффициенты каждого сегмента:
    for i in 0..n-1:
        a[i] = y[i]
        b[i] = (y[i+1]-y[i])/h[i] - h[i]*(m[i+1]+2*m[i])/6
        c[i] = m[i] / 2
        d[i] = (m[i+1] - m[i]) / (6*h[i])
```

### 4. RBF-интерполяция (Radial Basis Functions)

Для рассеянных данных в многомерном пространстве:

$$s(\mathbf{x}) = \sum_{i=1}^{n} \lambda_i \phi(\|\mathbf{x} - \mathbf{x}_i\|)$$

Типичные базисные функции:
- Гауссова: $\phi(r) = e^{-(\varepsilon r)^2}$
- Мультиквадратика: $\phi(r) = \sqrt{1 + (\varepsilon r)^2}$
- Тонкая пластина: $\phi(r) = r^2 \ln r$

> 📐 Применение: RBF используется для интерполяции геометрии деформированных поверхностей (например, изогнутый профиль зуба под нагрузкой).

## Сравнение методов

| Метод | Проходит через точки | Гладкость | Многомерность | Устойчивость |
|-------|---------------------|-----------|---------------|--------------|
| Лагранж | Да | $C^\infty$ (на отрезках) | Нет | Низкая (Рунге) |
| Кубич. сплайн | Да | $C^2$ | Нет | Высокая |
| B-Spline | Нет | $C^{k-2}$ | Нет | Высокая |
| RBF | Да | $C^\infty$ | Да | Зависит от $\varepsilon$ |
